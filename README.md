# MLP Memory Card Game

A memory game made with React for the Odin Project's React course! This is manly used for practicing useEffect hooks!

This project uses the [PonyAPI](https://ponyapi.net) for fetching the images of MLP characters and their names!

Unfortunately, the API is only http and not https, so there is no way for me to currently host it. I'm looking into fixing this issue currently.

## How to play

Click on as many cards without them being the same as you can! How many can you remember before clicking a card more than once?
